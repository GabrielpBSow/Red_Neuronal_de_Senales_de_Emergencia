# Red Neuronal de Señales de Emergencia

Proyecto desarrollado en Python utilizando TensorFlow y OpenCV para el reconocimiento de señales de emergencia mediante visión artificial y redes neuronales convolucionales (CNN).

## Características

- Entrenamiento de red neuronal convolucional.
- Clasificación de 7 tipos de señales.
- Uso de imágenes de entrenamiento y validación.
- Predicción de señales mediante imágenes.

## Tecnologías

- Python
- TensorFlow
- Keras
- NumPy
- Pillow

## Clases detectadas

- Alarma de Emergencia
- Alto Voltaje
- Cubrebocas
- No fumar
- Punto de Reunión
- Salida de Emergencia
- Sustancia Tóxica
